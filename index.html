The Varied Die Roller Of Mythic Proportions

Welcome. This is a work in progress. Both die rollers now work, including functionality to roll multiple dice and remove previous rolls. Recently added themes, including separate selection for just die. Currently working on how die rolls present themselves, inclduing displaying the dice with the set sides instead of just a circle. In the far future there will be monsters.

Put in the number of sides and number of dice.
Alternatively you can choose a preset die.
Monsters can be summoned here.
